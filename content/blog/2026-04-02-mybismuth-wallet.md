**myBismuth Wallet** is an open source wallet built to make Bismuth easy to access on virtually any device.
<!--more-->

### myBismuth Wallet

Designed as a **Progressive Web App (PWA)** and deployed at https://mybismuth.com, it gives users a simple way to install and use a full-featured Bismuth wallet directly from the browser, without relying on a traditional app store or heavy native software.

#### Compatibility
One of the wallet’s biggest strengths is its broad platform compatibility. Through the Chrome browser, myBismuth Wallet can be installed and used on Windows 11, macOS, Linux, Android, and iPhone, giving users a consistent experience across desktop and mobile environments. Once installed, it behaves much like a native application while keeping the flexibility and convenience of a modern browser-based deployment model.

#### Opensource
Because it is both open source and lightweight, myBismuth Wallet is especially well suited for users who value transparency, portability, and simplicity. The wallet avoids unnecessary complexity while still delivering the essential tools needed to manage assets on the Bismuth network. Its installable PWA design makes it fast to access, easy to maintain, and practical for everyday use across many types of devices.

#### Non-custodial
myBismuth Wallet is fully non-custodial, which means users remain in control of their wallet credentials at all times. The mnemonic is stored locally in the browser profile rather than being entrusted to a third party. Users can create a new wallet or import an existing 24-word seed phrase, making setup and recovery straightforward in a familiar browser-native workflow.

#### ED25519 address scheme
The wallet is designed for Bismuth’s ed25519 address scheme and supports multiple deterministic accounts derived from a single seed phrase. This allows users to manage several addresses within one wallet while keeping backup and recovery simple. For those who want additional flexibility, account-based private keys can also be derived with the BIS Paper Wallet and later imported into Tornado Wallet.

#### Bismuth Tokens
Beyond standard BIS transfers, myBismuth Wallet also supports Bismuth-based tokens. Users can view token balances, inspect token transaction history, and send supported tokens from the same interface. This makes it more than a basic wallet for BIS alone, extending its usefulness across the wider Bismuth ecosystem.

#### Interoperability
The generated seed phrases are compatible with the BIS-Paper wallet, allowing seamless recovery and interoperability between myBismuth Wallet and BIS-Paper. While myBismuth Wallet does not directly display private keys, they can still be accessed through BIS-Paper and, if needed, imported into the Tornado Bismuth Wallet.

#### Languages
Accessibility and usability are also central to the wallet’s design. The interface includes multiple language support, light and dark mode, and a responsive experience that works well on both desktop and mobile devices. These features help make the wallet approachable for newer users while still offering the functionality expected by more advanced participants in the network.

#### Key Benefits
At its core, myBismuth Wallet is about practical self-custody delivered in a modern, accessible form. It is open source, lightweight, and available across major desktop and mobile platforms as an installable Progressive Web App through the Chrome browser at https://mybismuth.com. For users who want an easy way to manage BIS and Bismuth-based tokens without giving up control, myBismuth Wallet offers a clean and capable solution.
